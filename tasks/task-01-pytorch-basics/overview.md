# Task 1: PyTorch Foundations

## Objective
Learn the essential PyTorch concepts needed to implement BERT: tensors, autograd, nn.Module, and DataLoader.

## Learning Outcomes
- Understand PyTorch tensors and operations
- Master automatic differentiation (autograd)
- Build custom neural network modules
- Use DataLoader for efficient batch processing

## Estimated Time
2 hours

## Reading Material
- [PyTorch Documentation](https://pytorch.org/docs/stable/index.html)
- [Deep Learning with PyTorch](https://pytorch.org/tutorials/beginner/deep_learning_60min_blitz.html)
